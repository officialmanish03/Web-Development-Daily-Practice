Task Manager Mini
The Task Manager Mini is a basic to-do list application that allows users to add, delete, and edit tasks dynamically. It’s a lightweight project designed to improve productivity while exploring core JavaScript functionalities.

📜 Features:
Dynamic Task List:

Add new tasks using an input field and "Add Task" button.
Tasks are displayed in a structured list format.
Task Removal:

Delete specific tasks with the "Delete" button.
Uses JavaScript's splice() method for efficient task removal.
Task Editing:

Modify tasks using the "Edit" button.
Save the updated task seamlessly with event listeners.
Interactive UI:

Clean, minimalist design with responsive elements for a smooth user experience.
JavaScript Functionalities:

Powered by core JavaScript for dynamic list management.
Event listeners for handling user interactions like adding, deleting, and editing tasks.
Responsive Design:

Adapts to various screen sizes, ensuring usability across devices.
Additional Enhancements:

Focused input field for seamless task entry.
Confirmation prompts for delete actions (future enhancement idea).

🔧 Technologies Used:
HTML: Provides the structure of the application.
CSS: Styles the page for a clean and modern look.
JavaScript: Implements the dynamic functionalities of the task manager.

🛠️ How to Use:
Clone or download the repository.
Open index.html in any browser.
Enter tasks in the input field and click "Add Task" to populate the list.
Use the "Edit" button to modify tasks or the "Delete" button to remove them.

🎯 Why This Project?
This project emphasizes the practical implementation of JavaScript array methods like splice() and showcases the importance of responsive and user-friendly design in web applications.
