Profile Card - Day 1
This Profile Card project is a fully responsive and interactive component built with HTML and CSS. Key highlights include:

Effective, responsive styling using rem units.
Tabs for About, Skills, and Projects, organized within a div layout.
Anchor tags for linking to social media handles.
Integrated Box Icons for icons and SVG files for the logo.
Styled with Google Fonts for a polished look.
Bootstrap framework for additional responsiveness and utility classes.
This card is a perfect example of a modern, clean, and functional design, offering a dynamic way to showcase personal profiles.
