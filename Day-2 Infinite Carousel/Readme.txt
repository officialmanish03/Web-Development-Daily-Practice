Infinite Carousel
This project is part of my Web Development Daily Practice challenge, where I create and upload a new mini-project daily for 30 days.

🎠 Project Overview
Infinite Carousel is a fully responsive photo carousel built entirely with HTML and CSS. It’s designed to showcase your best photos and memories in a unique and creative way.

✨ Features:
Smooth Infinite Scrolling: Utilizes CSS transition and transform properties for seamless animations.
Dynamic Design: Highlights photos with a clean and modern look.
Pure HTML & CSS: No JavaScript required for functionality.
Responsive Layout: Ensures compatibility across devices.
🔧 Technologies Used:
HTML: For structure and layout.
CSS: For styling, transitions, and animations.
🖼️ Project Highlights:
Showcasing three beautiful images in an infinite loop.
Creative use of transform and keyframes for smooth animation.
Minimalistic design with attention to responsiveness.
🚀 How to Use:
Clone or download the repository.
Open index.html in your browser to view the project.
Replace the placeholder images with your own to personalize the carousel.
🌟 Why This Project?
This project is a simple yet effective way to practice CSS animations and responsive design, while also providing a practical and visually appealing solution for photo showcases.
