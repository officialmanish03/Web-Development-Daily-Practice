🎧 Music Player
A simple music player built using HTML, CSS, and JavaScript that allows users to play, pause, and switch songs stored in their local folder. It also includes a volume control and a basic song progress bar for a better listening experience.

📜 Features:

🎵 Basic Music Controls:

Play/Pause button to start or stop the current song.
Next Song button to switch to the next available track.

🔊 Audio Controls:

Volume Control slider to adjust the sound level.
Progress Bar to visualize song playback (duration not displayed in this version).

🖥 JavaScript Functionalities:

Uses event listeners and DOM manipulation to control audio playback.
Interactive UI for smooth user experience.

🛠 Technologies Used:
HTML: Provides the basic structure of the music player.
CSS: Styles the player for a visually appealing design.
JavaScript: Implements core functionalities like play, pause, next song, and volume adjustment.

🎯 How to Use:
Clone or download the repository.
Add some downloaded songs to your local project folder.
Open index.html in a browser.
Click Play to start the music, Pause to stop, and Next to play the next track.
Adjust the volume slider for sound control.
